2-1.URI / URL / URN

2-1.URI / URL / URN

【URI（Uniform Resource Identifer）】

　→リソースを一意に識別するID

2-1.URI / URL / URN

URL（Uniform Resource Locator）

　→リソースの場所（住所）を示すもの

URN（Uniform Resource Name）

　→リソースの名前（恒久的なID）を示すもの


URLとURNを包括したものがURI。
